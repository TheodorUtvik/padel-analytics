# Padel Analytics

Hobby application to learn Data Science.  

## Environment

- `API_KEY` is required in `.env`.
- Optional:
  - `PADELAPI_BASE_URL` (default: `https://padelapi.org`)
  - `PADELAPI_TIMEOUT_SECONDS` (default: `30`)
  - `PADELAPI_OFFLINE=1` to force cached-only mode (reads from `data/raw/`)
